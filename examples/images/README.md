# Example Images

Drop example scene images here that users can copy to their Home Assistant.

These images will be copied to `/config/www/chameleon/` on the user's HA instance.

## More Recommendations

Go to Unsplash and search for images using keywords like:

- **Warm tones**: Sunset, autumn, fire
- **Cool tones**: Ocean, winter, night sky
- **Vibrant**: Neon lights, abstract gradients
- **Natural**: Forest, meadow, mountains
